# Versuch: Ultraschall Sensor

## Material
- 1x Arduino UNO
- 1x HC-SR04 Ultraschallsensor
- 4x Jumper Wire Male to Male
- 1x Laptop mit Software TODO
- 1x USB A zu B Kabel (Laptop zu Arduino UNO)
